markdown使用简介

一.markdowm是什么
markdown是一种简单的标记语言，简称md,它对应html的一些标签通过一些转换库把md转换成html,或把html转换成md。

二.md的作用
用来在网页上展示文章，省区排版布局的烦恼。

三.md的使用-基本语法
1.标题
"#"+空格键

2.段落
标题下另起一行，空格键+直接输入

3.有序表格
数字+空格

4.无需列表
"\"+空格

5.代码
"\"+3个"~"
中间写代码的内容
"\"+3个"~"

6.行内代码
~写代码的内容~

7.图片
![]+(图片地址)

8.链接
[链接文字]（http://jirengu.com）

9.引用
">"+"引用"

10.表格
表头
"│" + "表头内容" + “│"

虚线

"│" + "---" + "│"

内容

“│" + "内容" + "│"
